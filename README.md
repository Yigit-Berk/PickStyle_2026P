# PickStyle_2026P
Bir android uygulama fikrinin prototip olarak uygulandığı MVVM üzerine kurulu bir proje örneğidir.
Kullanıcının anlık olarak kombinlerini yerel veritabanında tutarak hızlı bir şekilde liste görünümünden erişebilmesi amaçlanmıştır.

# Technologies
Pek çok Android Jetpack kütüphanesine bu projede yer verildi.

Kullanılan teknolojiler ve yapılar: CameraX, MVVM, Room, SQlite, State Management, Clean Architecture, Jetpack Compose...
